# Submissions Directory

This directory contains all team project submissions for Techsprint hackathon.

## Structure

Each team should create a folder with their team name:

```
submissions/
├── team-alpha/
│   ├── README.md
│   ├── src/
│   └── ...
├── team-beta/
│   ├── README.md
│   ├── src/
│   └── ...
└── ...
```

## Submission Instructions

Please refer to [CONTRIBUTING.md](../CONTRIBUTING.md) for detailed submission guidelines.

## Important

- One folder per team
- Include all necessary files to run your project
- Do not modify other teams' folders
- Follow the naming convention: `team-name` (lowercase, hyphens for spaces)
